Написаны API - автотесты для сайта https://petstore.swagger.io/ с использованием
методов GET,PUT,POST,DELETE в блоке "pet".


Структура проекта

Settings - содержит настройку, для определения параметров запроса.
DeleteTest - содержит тест для метода DELETE.
GetTest - содержит тест для метода GET.
PostTest - содержит тест для метода POST.
PutTest - содержит тест для метода PUT.

Запуск тестов

Ввести в командной строке Windows:

mvn clean test
